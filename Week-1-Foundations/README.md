# Week 1 — Blockchain Foundations

> **Theme:** Understand how blockchains work at the protocol level, master Solidity, and deploy your first smart contract.

---

## Week 1 Goals

By end of this week you will:
- Explain how a blockchain works cryptographically (hashing, Merkle trees, digital signatures)
- Trace a transaction from wallet to block finality
- Explain PoW vs PoS at a deep level
- Write, test, and deploy smart contracts using Hardhat
- Build and deploy a fully functional ERC-20 token

## Week 1 Mini Project

**"LaunchPad Token"** — Deploy a real ERC-20 token with:
- Fixed supply of 1,000,000 tokens
- Owner-only minting
- Transfer restrictions during launch period
- Etherscan verified

---

## Key Concepts This Week

- Cryptographic hashing (SHA-256, Keccak-256)
- Merkle trees
- Digital signatures (ECDSA)
- The EVM (Ethereum Virtual Machine)
- Gas mechanics
- Solidity: storage, memory, calldata, mappings, events
