# Week 2 — DeFi & Automated Market Makers

> **Theme:** Understand how decentralized finance actually works under the hood — AMMs, liquidity pools, DEXs, lending, and staking. Build a working AMM.

---

## Week 2 Goals

By end of this week you will:
- Understand the x·y=k AMM formula and its implications
- Calculate price impact, slippage, and impermanent loss mathematically
- Know how Uniswap V2/V3 work under the hood
- Understand DeFi primitives: DEX, lending (Aave/Compound), staking, yield
- Build a complete React + Web3 frontend using Wagmi/Viem
- Implement a simple AMM contract from scratch

## Week 2 Mini Project

**"SimpleSwap"** — A minimal Uniswap V2 clone:
- Two-token liquidity pool
- Add/remove liquidity
- Swap tokens using x·y=k formula
- LP tokens representing pool share
- Frontend showing pool stats and swap UI

---

## Key Concepts This Week

- x·y=k invariant and what it means geometrically
- Price impact vs slippage
- Impermanent loss calculation
- LP tokens as fractional pool ownership
- Flash swaps
- Concentrated liquidity (Uniswap V3 innovation)
- Wagmi v2 + Viem for type-safe Web3 frontends
