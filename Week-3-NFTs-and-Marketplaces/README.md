# Week 3 — NFTs & Marketplaces

> **Theme:** Build a complete NFT ecosystem — minting, metadata on IPFS, and a full marketplace with listing, buying, and royalties.

---

## Week 3 Goals

- Deeply understand how NFTs work at the contract level
- Store metadata on IPFS using Pinata
- Build an NFT marketplace modeled on OpenSea
- Implement ERC-2981 royalty standard
- Build a complete frontend: mint, gallery, list, buy

## Week 3 Mini Project

**"ArtBlock Marketplace"** — A simple NFT marketplace:
- ERC-721 NFT collection with IPFS metadata
- Mint NFTs from the frontend
- List NFTs for sale
- Buy NFTs (with ETH)
- On-chain royalties (ERC-2981)
- Image gallery
