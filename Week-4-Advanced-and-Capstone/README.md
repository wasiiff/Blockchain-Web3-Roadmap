# Week 4 — Advanced Topics & Capstone Project

> **Theme:** Go deep on advanced blockchain concepts — Layer 2, security vulnerabilities, MEV, oracles — then build and deploy your capstone project.

---

## Week 4 Goals

- Understand Layer 2 scaling solutions (Optimism, Arbitrum, ZK-rollups)
- Know every major smart contract vulnerability and how to prevent them
- Understand MEV, flash loans, and the dark forest
- Use Chainlink oracles for real-world data
- Build and deploy a production-grade capstone project

## Capstone Project Options

Choose one (or combine):

### Option A: Mini Uniswap Clone
- Full AMM with factory pattern
- Multiple pools
- Router for multi-hop swaps
- Price oracle (TWAP)
- Frontend with swap UI

### Option B: NFT Marketplace (Production-Grade)
- ERC-721 + ERC-1155 support
- Auction system (English + Dutch)
- Offers/bids
- Bundle sales
- Off-chain orders

### Option C: DAO with Governance Token
- ERC-20 governance token (votes)
- Governor contract (on-chain proposals)
- Timelock for safety
- Treasury management
- Frontend for voting
